## Installation

LWEncryptor is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'LWEncryptor'
```

**Carthage**
```ruby
github "luowei/LWEncryptor"
```
